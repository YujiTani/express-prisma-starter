# express-prisma-starter

Express + React + Prisma を使用したフルスタック開発のスターターキット

## 技術スタック

### フロントエンド
- React
- TypeScript
- Vite

### バックエンド
- Express
- Prisma (ORM)
- PostgreSQL

### 開発環境
- Docker
- Biome (フォーマッター/リンター)
- Husky (Git hooks)

## 特徴

### コード品質管理
- Biomeによる一貫したコードスタイル
- コミット前の自動フォーマット
- プッシュ前の静的解析チェック

### Docker構成
- PostgreSQLコンテナ
- APIサーバーコンテナ
- フロントエンドコンテナ

### データベース
- PostgreSQL
- Prismaによる型安全なデータアクセス
- マイグレーション管理
